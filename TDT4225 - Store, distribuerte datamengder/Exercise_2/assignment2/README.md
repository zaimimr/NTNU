## How to run ⚡️

### Setup Docker
To setup a fresh Docker container simply run

```docker-compose up```

### Insert data 
```python3 tasks.py -i```

### Run all tasks
```python3 tasks.py -a```

### Run selected tasks
If you only want to run task 1 and 2

```python3 tasks.py 1 2```
